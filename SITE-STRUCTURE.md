# Oil-Immersed Transformer Temperature Control Website Structure

## Positioning

An English B2B inquiry website focused only on oil-immersed transformer temperature indication, control, and remote monitoring. The site should serve transformer manufacturers, utilities, EPC contractors, panel builders, distributors, and retrofit engineers.

Primary conversion: submit an RFQ with application and selection data.

## Internal linking

- Every product page links to its parent category, two relevant applications, the selection guide, wiring guide, and RFQ page.
- Every application page links to the suitable OTI, WTI, or monitoring device.
- Every technical guide includes a comparison/selection table and contextual links to product pages.
- Category pages compare models instead of repeating product descriptions.

## Product page template

1. Product name, model, one-sentence purchasing value, product image, RFQ CTA
2. Application fit and key benefits
3. Technical specification table
4. Available models and output options
5. Installation dimensions and drawings
6. Wiring / signal integration
7. Cooling, alarm, and trip logic
8. Customization options
9. Quality and test documentation
10. Selection checklist
11. Related products and resources
12. Compact RFQ form

## RFQ fields

- Transformer type and rating
- OTI, WTI, or integrated monitoring requirement
- Measuring range
- Number of adjustable switch contacts
- Required signal: Pt100 / 4–20 mA / RS485 / relay
- Power supply
- Capillary length
- Temperature bulb size and mounting thread
- Ambient temperature / altitude / special environment
- Existing model or drawing upload
- Quantity, destination country, required delivery date
